Frequently Asked Questions

Is this application free?
Yes this application will be free, all you need to get started today is an email account to create an account!
Does Pack and go have packages for every city across the world?
No, Pack and Go will not have a package for EVERY city, however users are adding new packages each day. So if a city searched does not already have a package listed, you could be the first to create one.
Can any unauthorized user edit my package?
Packages will have the option to be set to private or public. If the package is public, users will be able to add them and customize them further as their own package.
What if I want to delete my account?
Send a request on our contact page with your information for the request and an Administrator will contact you on the next steps.
Is there a rating system for the multiple packages?
Yes, packages will have a rating system which each city searched can be sorted by rating. You can also connect with other users with the comment section on any ramarks you may have on the package.
